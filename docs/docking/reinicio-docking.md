# 🔌 Reinicio del docking station

Después de un apagón, el docking station puede quedar con energía residual o con una conexión inestable. Eso puede hacer que los monitores, los puertos USB o la red no respondan como deberían.

Antes de empezar: 
Si algo no funciona, no fuerces los cables. Esta guía está pensada para corregir fallas comunes después de un corte de energía.


## Qué vas a corregir

Restablecer el docking para recuperar monitores, USB y red después de un apagón.

## Paso a paso

1. Identifica el cable de alimentación.
   Busca el **cable redondo de alimentación** del docking station. En muchos modelos, el conector tiene una **luz azul** cuando está conectado, en algunos casos la luz es blanca o en muy pocos, no tienen luz.

   ![Parte trasera del docking mostrando el conector de alimentación](/img/docking.png)
   *Parte trasera del docking mostrando el conector de alimentación.(ultimo de izquierda a derecha)*

2. Desconecta el cable de alimentación del docking station y, si aplica, también retira el adaptador de corriente de la toma eléctrica.

3. Con el docking station sin energía, mantén presionado el botón de encendido durante **15 segundos**. Esto ayuda a liberar energía residual y a reiniciar el estado interno del equipo.

Evita confusiones: 
No desconectes cables USB ni cables de video en este paso. Solo retira la alimentación del docking station.


4. Conecta nuevamente el cable de alimentación y espera unos segundos para que el docking station termine de iniciar.

## Verificación final

Confirma que estos elementos vuelvan a funcionar:

- [ ] Monitores
- [ ] USB
- [ ] Red

Si alguno sigue sin responder, revisa que el docking station esté bien conectado y repite el procedimiento una sola vez, si el problema persiste contacta a TI.
