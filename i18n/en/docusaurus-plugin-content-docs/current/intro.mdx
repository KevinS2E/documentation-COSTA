# IT Support Guide

Internal guide for company users, designed to help solve common issues after power outages without needing technical knowledge.

This documentation helps you:

- Restart the **docking station**
- Restart the **laptop**
- Recover monitors, USB, and network when possible

Quick use
If the problem started after a power outage, begin with the docking station guide. It is usually the fastest and safest step.

## What is this guide about?

This documentation explains what to do when a power outage causes issues with your work equipment. The instructions are written so that anyone can follow them calmly and without confusion.

## When should you use this guide?

Use this guide when any of these situations happen:

- The monitors do not work
- The laptop does not respond
- The USB ports do not work
- The network is unavailable
- The problem started after a power outage

## What will you learn?

By following this documentation, you will be able to:

- Restart the docking station correctly
- Restart the laptop without affecting other cables
- Solve basic issues without asking IT for immediate help
- Know when it makes sense to escalate the case to technical support

## Recommended flow

1. Restart the docking station
2. If that does not work, restart the laptop
3. If the problem continues, contact IT support

When to contact IT support
If after following both guides the equipment still does not respond, or if you notice a burning smell, damaged cables, or any sign of electrical risk, stop the process and contact IT.
